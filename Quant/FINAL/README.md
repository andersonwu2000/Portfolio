# 台股營收動能選股訊號：回測驗證

本專案以自建的回測程式，驗證「營收成長動能」作為台股選股訊號的有效性與可行性。
研究目的：**如何判斷一個選股訊號是否具備真實的預測力。**

## 專案結構

| 目錄 | 內容 |
|------|------|
| `Report/` | 研究成果：圖表與（日後補上的）研究報告。 |
| `Backtest/` | 自建的回測程式與策略邏輯。 |
| `Analysis/` | 各項研究分析的程式碼。 |
| `Data/` | 約 1900 支台股的價格與月營收（含已下市公司，以避免倖存者偏差）。 |

## 回測方法

為縮小回測與實盤的差距，回測程式有幾項特點：

- **只選當下買得到的股票**：每個換股日重新認定可投資範圍（須有成交、上市夠久、具流動性），並保留已下市公司，避免只看活下來的贏家而高估績效（倖存者偏差）。
- **嚴格防未來函數**：策略在任一時間點只能使用當時已公開的資訊；台股月營收延後 40 天才納入，杜絕用到尚未公布的數字。
- **貼近真實的交易成本**：依台股實況不對稱扣費——買進約 0.17%、賣出約 0.47%（含 0.3% 證交稅與約 0.03% 滑價）；高換手策略因賣出較多而付出更高成本。
- **只做多、不放空**：台股放空成本高且受限，策略僅以做多形式部署，回測不假設能自由放空。

（各點的詳細說明見 `Backtest/`。）
