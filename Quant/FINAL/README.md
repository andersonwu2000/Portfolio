# 台股營收動能選股訊號：回測驗證

本專案以自建的回測程式，驗證「營收成長動能」作為台股選股訊號的有效性與可行性。
研究的核心是：**如何判斷一個選股訊號是否具備真實的預測力。**

本文件說明專案的組成與閱讀方式。

## 專案結構

| 目錄 | 內容 |
|------|------|
| `Report/` | 研究成果：圖表與（日後補上的）研究報告。讀者從這裡開始。 |
| `Backtest/` | 自建的回測程式與策略邏輯。 |
| `Analysis/` | 各項研究分析的程式碼。 |
| `Data/` | 約 1900 支台股的價格與月營收（含已下市公司，以避免倖存者偏差）。 |

## 閱讀順序

1. **圖表**（`Report/`）——各項分析的視覺化結果，最快掌握全貌。
2. **研究分析**（`Analysis/`）與**回測程式**（`Backtest/`）——個別主題的程式碼與進一步細節。

> 每個子目錄皆附有 README，說明其組成。
